# 2026 暑期 Python 课程项目

## 项目列表

### Project 1 — Python 基础练习
入门级 Python 编程练习，涵盖变量、流程控制、函数、循环等基础语法，共 14 个练习文件（编号 1.1 ~ 6.2）。

### Project 2 — 机器学习回归预测
基于 scikit-learn 的回归模型项目，包含特征重要性分析与预测结果可视化。  
`main.py` — 训练 + 预测全流程  
`model.pkl` — 训练好的模型  
`images/` — 特征重要性图与预测散点图

### Project 3 — 石头剪刀布手势识别 (YOLO11n)
基于 YOLO11n 的目标检测项目，使用 Roboflow 上 15,874 张图片的数据集，训练识别剪刀、石头、布三种手势。  
支持单张图片推理和实时摄像头检测。验证集 mAP50 达到 **0.940**，测试集 mAP50 达到 **0.934**。

详细说明见各子目录 README。
