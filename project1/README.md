# Project 1：Python 基础练习

入门级 Python 编程练习，涵盖变量、数据类型、运算符、流程控制、函数、组合数据类型等基础语法，共 14 个练习文件。

## 项目结构

```
project1/
├── README.md           # 本文件
├── 1.1.py              # 输入正整数，输出其平方
├── 1.2.py              # 按模板输出学生录取信息
├── 1.3.py              # 求直角三角形斜边长
├── 2.1.py              # 计算两点间距离
├── 2.2.py              # 三位正整数各位立方和
├── 2.3.py              # 统计英文字母出现次数（不区分大小写）
├── 3.1.py              # 成绩等级判定
├── 3.2.py              # 字符类型判断
├── 4.1.py              # 判断整数位数
├── 4.2.py              # 计算 1! + 2! + … + 10!
├── 5.1.py              # 函数求斜边长度
├── 5.2.py              # 求 n 以内被 3 和 7 整除的数之和的平方根
├── 6.1.py              # 生成大写英文字母列表
└── 6.2.py              # 根据学号查询年级和学院
```

## 运行方式

所有文件均为独立运行的脚本，无需额外安装第三方库：

```bash
cd project1
python 1.1.py
```

每个脚本读取标准输入（`input()`），输出到标准输出（`print()`）。

## 环境要求

- Python 3.6+（推荐 3.12+）
- 无需第三方依赖（仅使用标准库和 `math`）
