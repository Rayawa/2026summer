### （一）数据类型

#### 1) 输入正整数，输出其平方

**题目要求**：输入一个正整数，输出对应的平方。例如输入 5，输出 25。

**代码实现**（`project1/1.1.py`）：

```python
n = int(input())
print(n * n)
```

**运行示例**：
```
输入：5
输出：25
```

**解题思路**：使用 `int()` 将输入转换为整数，直接计算 n² 即可。

#### 2) 按模板输出学生录取信息

**题目要求**：按模板"×××同学，你已被我校×××学院×××专业录取，请于××年××月××日准时到我校报到"输出列表内容。

```
王若飞同学，你已被我校经济管理学院会计专业录取，请于2020-9-7准时到我校报到
```

#### 3) 求直角三角形斜边长

**题目要求**：已知直角三角形两条直角边，求斜边长度。

**代码实现**（`project1/1.3.py`）：

```python
a, b = map(float, input().split(','))
print((a * a + b * b) ** 0.5)
```

**运行示例**：
```
输入：3,4
输出：5.0
```

**解题思路**：根据勾股定理 c = √(a² + b²)，使用 `**0.5` 计算平方根。

### （二）运算符和表达式

#### 1) 计算两点间距离

**题目要求**：输入两个点的坐标，计算两点间距离，精确到小数点后两位。

**代码实现**（`project1/2.1.py`）：

```python
x1, y1 = eval(input('输入第一个点的坐标，用逗号分隔：'))
x2, y2 = eval(input('输入第二个点的坐标，用逗号分隔：'))
p = ((x1 - x2) ** 2 + (y1 - y2) ** 2) ** 0.5
print('点({},{})和点({},{})之间的距离是{:.2f}'.format(x1, y1, x2, y2, p))
```

**运行示例**：
```
输入第一个点的坐标，用逗号分隔：0,0
输入第二个点的坐标，用逗号分隔：3,4
点(0,0)和点(3,4)之间的距离是5.00
```

**解题思路**：使用 `eval()` 解析逗号分隔的坐标输入，根据欧几里得距离公式计算，`{:.2f}` 保留两位小数。

#### 2) 三位正整数各位立方和

**题目要求**：输入一个三位正整数，输出各位数字的立方之和。

**代码实现**（`project1/2.2.py`）：

```python
n = int(input())
a = n // 100
b = (n // 10) % 10
c = n % 10
print(a ** 3 + b ** 3 + c ** 3)
```

**运行示例**：
```
输入：312
输出：36
```

**解题思路**：通过整除和取余运算分解百位、十位、个位数字，分别立方后求和。

### （三）选择结构

#### 1) 成绩等级判定

**题目要求**：输入学生成绩，输出对应等级，并验证成绩有效性。
- [90~100]：优秀
- [80~90)：良好
- [70~80)：中等
- [60~70)：及格
- [0~60)：不及格
- 无效成绩（<0 或 >100）：提示无效

**代码实现**（`project1/3.1.py`）：

```python
score = float(input())
if score < 0 or score > 100:
    print('无效')
elif score >= 90:
    print('优秀')
elif score >= 80:
    print('良好')
elif score >= 70:
    print('中等')
elif score >= 60:
    print('及格')
else:
    print('不及格')
```

**运行示例**：
```
输入：85
输出：良好

输入：-5
输出：无效
```

**解题思路**：使用 if-elif-else 多分支结构，先验证成绩范围，再从高到低依次判断等级。

---

#### 2) 字符类型判断

**题目要求**：输入一个字符，判断并输出是数字字符、大写字母、小写字母还是其他字符。

**代码实现**（`project1/3.2.py`）：

```python
ch = input()
if ch.isdigit():
    print('这是一个数字字符')
elif ch.isupper():
    print('这是一个大写英文字符')
elif ch.islower():
    print('这是一个小写英文字符')
else:
    print('这是其他字符')
```

**运行示例**：
```
输入：A
输出：这是一个大写英文字符

输入：5
输出：这是一个数字字符
```

**解题思路**：利用 Python 字符串的 `isdigit()`、`isupper()`、`islower()` 方法判断字符类型。

### （四）循环结构

#### 1) 判断整数位数

**题目要求**：输入一个整数，判断并输出它是几位数。

**代码实现**（`project1/4.1.py`）：

```python
x = int(input())
n = 0
y = x
while x:
    n += 1
    x //= 10
print('{}是{}位数'.format(y, n))
```

**运行示例**：
```
输入：32767
输出：32767是5位数
```

**解题思路**：使用 `while` 循环反复整除10，每除一次计数加1，直到数字变为0。注意保存原始值用于输出。

---

#### 2) 计算 1! + 2! + … + 10!

**题目要求**：计算 1! + 2! + 3! + … + 10!。

**代码实现**（`project1/4.2.py`）：

```python
s = 0
for i in range(1, 11):
    s += __import__('math').factorial(i)
print(s)
```

**运行结果**：
```
4037913
```

**解题思路**：利用 `math.factorial()` 函数计算每个数的阶乘并累加求和。

### （五）函数

#### 1) 函数求斜边长度

**题目要求**：定义函数 fun，根据两条直角边计算斜边长度。

**代码实现**（`project1/5.1.py`）：

```python
def fun(x, y):
    return (x ** 2 + y ** 2) ** 0.5

leg1, leg2 = eval(input('输入两条直角边，以逗号分隔：'))
print('斜边的长度为：{:.2f}'.format(fun(leg1, leg2)))
```

**运行示例**：
```
输入两条直角边，以逗号分隔：3,4
斜边的长度为：5.00
```

**解题思路**：将勾股定理计算封装为函数，便于复用。

---

#### 2) 求 n 以内被 3 和 7 整除的数之和的平方根

**题目要求**：编写函数 fun，求 n 以内（不包括 n）同时能被 3 与 7 整除的所有自然数之和的平方根 s。

**代码实现**（`project1/5.2.py`）：

```python
def fun(n):
    s = 0
    for i in range(1, n):
        if i % 3 == 0 and i % 7 == 0:
            s += i
    return s ** 0.5
```

**解题思路**：遍历 1 到 n-1，使用 `%` 运算判断是否同时被 3 和 7 整除（即被 21 整除），符合条件的数累加后返回平方根。

### （六）列表、元组与词典

#### 1) 生成大写英文字母列表

**题目要求**：生成元素均为 26 个大写英文字母的列表。

**代码实现**（`project1/6.1.py`）：

```python
ls = [chr(x) for x in range(65, 91)]
print(ls)
```

**运行结果**：
```
['A', 'B', 'C', ... , 'X', 'Y', 'Z']
```

**解题思路**：使用列表推导式，ASCII 码 65~90 对应大写字母 A~Z，`chr()` 将 ASCII 码转换为字符。

#### 2) 根据学号查询年级和学院

**题目要求**：输入学生学号，输出年级和学院信息。学号规则：4位入学年份 + 2位学院代码 + 1位专业代码 + 1位班级代码 + 2位序号。学院代码存储在字典中。

**代码实现**（`project1/6.2.py`）：

```python
xydm = {'01': '经济管理学院', '02': '机电学院', '03': '生命科学院',
        '04': '外国语学院', '05': '理学院', '06': '文法学院', '08': '计算机学院'}
sid = input()
year = sid[:4]
col = xydm.get(sid[4:6], '未知学院')
print(year, col)
```

**运行示例**：
```
输入：20240801001
输出：2024 计算机学院
```

**解题思路**：使用字符串切片提取学号各字段，通过字典的 `get()` 方法查询学院名称，设置默认值避免 KeyError。
